## v4 治理層｜AI × 心理服務的整合實踐

暖流人心將 AI 治理層整合進日常運作，協助臨床工作之外的內容、品質、流程治理。整合精神是「AI 是工具，臨床判斷是主場」。

### 內部運作架構（簡述）
- **戰略部**：對外整合、風險控管、跨領域決策
- **內容部**：心理普及內容、跨媒體素材、視覺資產
- **守護部**：SEO 經營、品質審核、法規對齊
- **整合運作**：四方合議場（三部 + 創辦人）

### 治理紀律
- 內容客製化、禁止模板套用
- 普查不抽查、所有產出逐項核對
- 法規邊界優先於效率
- 創辦人量能保護優先於進度

### 公開資源（陸續上線）
- 心理普及文章 markdown 版（連結至 .tw 主站正文）
- 心理服務詞彙表中英對照
- 台灣心理健康公益資源整理

## 業倫理聲明｜Ethics Statement

本 GitHub Pages 嚴格遵守台灣心理師業倫理規範與相關法規：

### 三條紅線
1. **絕不公開個案資料**（即使去識別化、即使 private repo）
2. **不公開治療師私人聯絡資訊**
3. **不誇大療效宣稱、不發布個案見證、不用招攬語**

### 法規對齊
- 個資法（§6 特種個資 / §19 §20 蒐集處理 / §28 損害賠償）
- 心理師法第 27 條廣告限制
- 心理師法第 13/14/18 條業務範圍
- 中華民國臨床心理師公會全國聯合會（ATCP）業倫理規範第 16 條保密義務
- 刑法第 318 條業務上知悉他人秘密罪
- 消費者保護法第 22 條真實性原則
- 著作權法

### 責任聲明
本網站所有內容為**心理普及教育用途**，不取代正式心理諮詢、心理治療或醫療診斷。如有心理健康需求，請至官網表單預約 [warmflowpsy.tw/booking](https://warmflowpsy.tw/booking) 或撥打安心專線 1925。
