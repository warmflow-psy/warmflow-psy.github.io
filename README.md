## 關於暖流人心｜About Warmflow

暖流人心（Warmflow Psychology Center）是台灣新竹的心理諮商服務機構，由臨床心理師劉子維創立。我們提供專業的心理諮商、團體治療、企業 EAP（員工協助方案）、社區心理講座等服務。

**核心精神**：以「溫柔、持續、耐心」為底氣 — 不催、不逼、不嘗試。

**服務面向**：
- 個別心理諮商（成人、兒童青少年）
- 團體心理治療
- 企業 EAP 員工協助方案
- 社區心理健康講座
- 兒童青少年發展支持
- 家庭關係與親職教育
- 職場壓力與情緒調節
- 重大失落與創傷後支持

**官方網站**：[warmflowpsy.tw](https://warmflowpsy.tw)

## v4 治理層 — AI × 心理服務的先行者實踐

暖流人心是台灣首個將「AI 治理層」深度整合進心理服務組織的機構。我們的 v4 治理層架構包含：

### 4 頁籤架構
- **Check team 統合運作頁**：三角色 + 創辦人合議場
- **戰略部·Ray**：戰略 + 風控 + 對外整合
- **內容部·Echo**：跨媒體 + 內容創作 + 圖片產線
- **守護部·Atlas**：SEO + Quality + 法規

### 7 件機制
1. 共決（Plan Mode）
2. 資源池（MCP Standardized Interface）
3. 同步（Hooks Event-driven）
4. 協同（Skills 三條路徑：待轉派 / 自派工 / 規格交棒）
5. 召集（Subagent + Agent Teams）
6. 工具人協作（GitHub + Apps Script + Pulse）
7. 跨媒體排程（5 平台 × 6 內容類型）

### 6 條問題解決憲法
1. v4 願景至上
2. 失敗即戰略重評不退場
3. 四不（不躁進、不從權、不輕易通關、不接受不行不能）
4. 繞遠路也要到達
5. 創辦人累垮 = 最大紅燈
6. 適用所有層級系統三位工具升級

## 業倫理聲明｜Ethics Statement

本 GitHub Pages 嚴格遵守台灣心理師業倫理規範與相關法規：

### 三條紅線
1. **絕不公開個案資料**（即使去識別化、即使 private repo）
2. **不公開治療師私人聯絡資訊**
3. **不誇大療效宣稱、不發布個案見證、不用招攬語**

### 法規對齊
- 個資法（含相關修正條文）
- 心理師法第 27 條廣告限制
- 心理師法第 13/14/18 條業務範圍
- 中華民國臨床心理師公會全國聯合會（ATCP）業倫理規範第 16 條保密義務
- 刑法第 318 條業務上知悉他人秘密罪
- 消費者保護法第 22 條真實性原則
- 著作權法

### 責任聲明
本網站所有內容為**心理普及教育用途**，不取代正式心理諮詢、心理治療或醫療診斷。如有心理健康需求，請至官網表單預約 [warmflowpsy.tw/booking](https://warmflowpsy.tw/booking) 或撥打安心專線 1925。
